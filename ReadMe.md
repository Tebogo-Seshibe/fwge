# fwge
FWGE WebGL Game Engine
Web based game engine library, built on top of the WebGL API

## Prerequisites
### Global
- node: >= 14.0.0
- pnpm: >= 9.1.3
- rimraf
- turbo
- typescript
### Local
- jsdom
- vitest

## Setup
```bash
pnpm clean # Only required to reset the project
pnpm install
pnpm build
```
